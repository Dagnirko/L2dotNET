L2dotnet

A server emulator for Lineage2 Interlude written in C#

Welcome to L2dotNET.

L2dotNET is a open source server emulator for the game Lineage2 C6 completely written in C#. If you like the idea of modernizing the game and moving away for the already existing oldfashioned l2j implementation then feel free to fork the project and contribute.
